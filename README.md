# Python Automation Script

## Description
This project automates file management tasks using Python.

## Features
- Rename files automatically
- Sort files into folders
- Generate log file
- User input support
- Exception handling

## Technologies Used
- Python
- os module
- shutil
- logging

## How to Run
1. Clone the repository.
2. Run:
   python automation.py
3. Enter the folder path when prompted.
